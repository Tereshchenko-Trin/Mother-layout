# Mother-layout
A one-page landing page, designed according to a [layout](https://www.figma.com/design/SHJTXIYNdV5ez7VpJtkdXD/Mother-layout?node-id=0-1&t=Axc8cWXEaesN6lca-1). You can [watch it now](https://tereshchenko-trin.github.io/Mother-layout/).

# Implemented Functionality
The following tasks were completed as part of the technical specifications:
- **Page Layout:** A full page layout was implemented based on the layout, adhering to the pixel-perfect principle.
- **Technologies:** Modern HTML5 and CSS3 standards were used.
- **Semantics:** Semantic tags were applied to improve accessibility and SEO.
- **CSS Methodology:** Styles are organized according to the BEM (Block, Element, Modifier) ​​methodology.
- **Adaptability:** A responsive layout was implemented to ensure correct page display on various devices (mobile, tablet, desktop).
- **Validation and Checks:** The code undergoes automatic quality checks (Lighthouse, W3C Validator, WAVE).
